Mermaid Diagrams

Note: Emanote documentation is a work-in-progress. Markdown sources for this site can be found here.

Mermaid lets you create diagrams and visualizations using text and code. You can define these diagrams in your Markdown code blocks.

  • Add the following to your page.headHtml, either in frontmatter or index.yaml (see YAML configuration)
    page:
  headHtml: |
    <script src="https://cdn.jsdelivr.net/npm/mermaid/dist/mermaid.min.js"></script>
    <script>
      mermaid.initialize({startOnLoad:false});
      mermaid.init(undefined,document.querySelectorAll(".mermaid"));
    </script>
  • Add a code block with mermaid language
    ```mermaid
%%{init: {'theme': 'forest', "flowchart" : { "curve" : "basis" } } }%%
graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;
```

The result will be: 

%%{init: {'theme': 'forest', "flowchart" : { "curve" : "basis" } } }%%
graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;

Note that mermaid.js will have trouble operating with PrismJS Syntax Highlighting.
